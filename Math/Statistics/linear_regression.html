線形回帰

線形回帰は，ある目的データを別のいくつかの観測データの線形和として表すことで，両者の関係性を説明し，新たに得られた観測データを元に未知の目的データを推定する用途で用いるモデルです．

線形単回帰モデル

線形単回帰モデルとは，2つのデータ変数 (xxx, yyy) について，散布図をプロットし，両者の間の線形な関係 (直線) を求めるモデルです．

統計の基礎で既にふれているように，その基本は中学校あたりで学習済みです．

Pythonコード 
# ランダムな線形データの生成
X, y, coef = make_regression(n_samples=100, n_features=1, noise=10, coef=True, random_state=42)

# 線形単回帰モデルの作成とトレーニング
model = LinearRegression()
model.fit(X, y)

# モデルによる予測
y_pred = model.predict(X)

# 図のプロット
plt.figure(figsize=(8, 6))
plt.scatter(X, y, color='blue', label='Data')
plt.plot(X, y_pred, color='red', label='Regression Line')
plt.title('Linear Regression', fontsize=20)
plt.xlabel('Feature', fontsize=15)
plt.ylabel('Target', fontsize=15)
plt.legend()
plt.grid(True)
plt.savefig('../figures/linear.png')

この図では，てきとうに作成したデータ点に対して単回帰を行い，得られた線 (回帰線) を赤で引いてあります．良い感じですね．たしかにそれっぽいところに線が引かれています．

概要はこの程度で十分なので，実際にモデルの中身を見ていきます．

まず，単回帰の場合は説明変数 xxx, 目的変数 yyy について

y=β0+β1x\begin{align} y = \beta_0 + \beta_1 x \end{align}y=β0​+β1​x​​

というシンプルな関係性を仮定します．ここで β0,1\beta_{0,1}β0,1​ は求める直線の切片，傾きに相当する変数です．特に β1\beta_1β1​ を回帰係数といいます．発想は初等数学でやる y=ax+by=ax+by=ax+b の関係式と同じです．回帰の目的は，データを最も上手く説明できるような (β0,β1)(\beta_0, \beta_1)(β0​,β1​)の組を探すことにあります．

ただ，実際には測定の誤差や，今回考えていない他の要因 (x2,3,...x_{2,3,...}x2,3,...​) の影響もあって式(1)が綺麗に成り立つことは考えにくいです．毎回多少ずれ ϵ\epsilonϵ が生じると考えます．

従って線形単回帰モデルはデータの組 (標本) の総数を nnnとした時，

yi=β0+β1xi+ϵi\begin{align} y_i = \beta_0 + \beta_1 x_i + \epsilon_i \end{align}yi​=β0​+β1​xi​+ϵi​​​

という形になります．ここで, i=1,...,ni.i.d.i = 1,...,n \quad i.i.d.i=1,...,ni.i.d. です．

言葉にすると，得られた iii 番目のデータの目的変数 yiy_iyi​ は，だいたい説明変数 xix_ixi​ の β1\beta_1β1​ 倍に定数 β0\beta_0β0​ を足した値で，誤差 ϵi\epsilon_iϵi​ だけずれている，というモデルです．

回帰係数の推定

では次に，得られた線形単回帰モデルの β0,1\beta_{0,1}β0,1​ を推定するアルゴリズムについて触れていきます．メジャーどこは大きく2種類あり，最小二乗法と最尤法があります．

最小二乗法

最小二乗法では，

「データに良く当てはまるモデルは，誤差 ϵi\epsilon_iϵi​が全ての iii について小さいモデルである」

と考えます．尤もらしいですね．ちなみに，ϵi\epsilon_iϵi​ については以下の仮定を設けることが多いです．

ϵ1,...,ϵni.i.d.E(ϵi)=0V(ϵi)=σ2\begin{align} \epsilon_1,...,\epsilon_n \quad \text{i.i.d.}\\ E(\epsilon_i)=0\\ V(\epsilon_i)=\sigma^2 \end{align}ϵ1​,...,ϵn​i.i.d.E(ϵi​)=0V(ϵi​)=σ2​​

誤差はデータごとに独立で，平均0，分散 σ2\sigma^2σ2 の同一分布に従うということです．σ2\sigma^2σ2 は未知のパラメータなので，必要に応じて推定する必要はあります．

これにより，同時に

y1,...,yni.i.d.E(yi)=β0+β1xiV(yi)=σ2\begin{align} y_1,...,y_n \quad \text{i.i.d.}\\ E(y_i)=\beta_0 + \beta_1 x_i\\ V(y_i)=\sigma^2 \end{align}y1​,...,yn​i.i.d.E(yi​)=β0​+β1​xi​V(yi​)=σ2​​

となります．

では本題です．

最小二乗法とは，予測値 yi^\hat{y_i}yi​^​ と実測値 yiy_iyi​ の差の全データでの二乗和を最小化するパラメータを探索する方法を意味し，今回のモデルについて考えると

arg min⁡β0,β1S=∑i=1nϵi2=∑i=1n{yi−(β0+β1xi)}2\argmin_{\beta_0,\beta_1} S = \sum_{i=1}^n \epsilon_i^2 = \sum_{i=1}^n \{ y_i - (\beta_0 + \beta_1 x_i) \}^2β0​,β1​argmin​S=i=1∑n​ϵi2​=i=1∑n​{yi​−(β0​+β1​xi​)}2

という問題になります．

arg min⁡β0,β1S\argmin_{\beta_0,\beta_1} Sargminβ0​,β1​​Sは，関数 SSS が最小値を取るような変数 β0,β1\beta_0, \beta_1β0​,β1​ の集合という意味で，つまりここでは二乗和 (SSS) を最小化するような β0,β1\beta_0, \beta_1β0​,β1​ の組み合わせを探すという意味です．実際の yiy_iyi​ の値と，xix_ixi​ の一次関数で表してみた予測値 yi^\hat{y_i}yi​^​ との誤差を全てのデータ iii について足し合わせたものが二乗和 SSS です．二乗しているのは符号の影響を打ち消すためです．ずれはプラスにもマイナスにもありますからね．

arg min⁡β0,β1S\argmin_{\beta_0,\beta_1} Sargminβ0​,β1​​S はただの問題設定であって，即座に計算できるものではありません．よってこの問題の解法を考えます．

変数それぞれについて，関数 SSS の最小値を取るわけですから，変数 (β0,β1\beta_0, \beta_1β0​,β1​) で偏微分しましょう．すると

∂S(β0,β1)∂β0=−2∑i=1n(yi−β1xi−β0)∂S(β0,β1)∂β1=−2∑i=1nxi(yi−β1xi−β0)\begin{align} \frac{\partial S(\beta_0, \beta_1)}{\partial \beta_0} = -2 \sum_{i=1}^n (y_i - \beta_1 x_i -\beta_0)\\ \frac{\partial S(\beta_0, \beta_1)}{\partial \beta_1} = -2\sum_{i=1}^n x_i(y_i -\beta_1 x_i- \beta_0) \end{align}∂β0​∂S(β0​,β1​)​=−2i=1∑n​(yi​−β1​xi​−β0​)∂β1​∂S(β0​,β1​)​=−2i=1∑n​xi​(yi​−β1​xi​−β0​)​​

となります．で，こいつらの最小値を求めるわけなのでそれぞれを =0=0=0 として約分したり移項したりして整理すると

β1∑i=1nxi+β0n=∑i=1nyiβ1∑i=1nxi2+β0∑i=1nxi=∑i=1nxiyi\begin{align} \beta_1 \sum_{i=1}^n x_i + \beta_0 n = \sum_{i=1}^n y_i \\ \beta_1 \sum_{i=1}^n x_i^2 + \beta_0 \sum_{i=1}^n x_i = \sum_{i=1}^n x_i y_i \end{align}β1​i=1∑n​xi​+β0​n=i=1∑n​yi​β1​i=1∑n​xi2​+β0​i=1∑n​xi​=i=1∑n​xi​yi​​​

が求められます．

式(12)を nnn で割ると，

β1∑i=1nxin+β0=∑i=1nyinβ1xˉ+β0=yˉ∴β0^=yˉ−β1^xˉ\frac{\beta_1\sum_{i=1}^n x_i}{n} + \beta_0 = \frac{\sum_{i=1}^n y_i}{n}\\ \beta_1 \bar{x} + \beta_0 = \bar{y}\\ \therefore \hat{\beta_0} = \bar{y} - \hat{\beta_{1}} \bar{x}nβ1​∑i=1n​xi​​+β0​=n∑i=1n​yi​​β1​xˉ+β0​=yˉ​∴β0​^​=yˉ​−β1​^​xˉ

となります．xˉ\bar{x}xˉは xxx の平均でしたね．さて，さらにこいつを先程の1つ目の式に代入して同様に nnn で割ると

途中式

β1∑i=1nxi2+(−β1xˉ+yˉ)∑i=1nxi=∑i=1nxiyiβ1(∑i=1nxi2n−xˉ∑i=1nxin)=∑i=1nxiyin−yˉ∑i=1nxina(x2ˉ−xˉ2)=xyˉ−yˉxˉ\beta_1\sum_{i=1}^n x_i^2 + (-\beta_1\bar{x} + \bar{y})\sum_{i=1}^n x_i = \sum_{i=1}^n x_i y_i \nonumber\\ \beta_1(\frac{\sum_{i=1}^n x_i^2}{n} - \bar{x}\frac{\sum_{i=1}^n x_i}{n}) = \frac{\sum_{i=1}^n x_i y_i}{n} - \bar{y}\frac{\sum_{i=1}^n x_i}{n}\\ a(\bar{x^2} - \bar{x}^2) = \bar{xy} - \bar{y}\bar{x}\\β1​i=1∑n​xi2​+(−β1​xˉ+yˉ​)i=1∑n​xi​=i=1∑n​xi​yi​β1​(n∑i=1n​xi2​​−xˉn∑i=1n​xi​​)=n∑i=1n​xi​yi​​−yˉ​n∑i=1n​xi​​a(x2ˉ−xˉ2)=xyˉ​−yˉ​xˉ

最尤法

次は最尤法での求め方です．最尤法では，誤差 ϵi\epsilon_iϵi​ が互いに独立に正規分布 N(0,σ2)N(0,\sigma^2)N(0,σ2) に従うことを要請します．

このとき，yiy_iyi​ も相互独立に正規分布 N(β0+β1xi,σ2)N(\beta_0 + \beta_1 x_i, \sigma ^2)N(β0​+β1​xi​,σ2) に従うことがわかるので， yiy_iyi​ の確率密度関数は

f(yi;β0,β1,σ2)=12πσ2exp⁡[−{yi−(β0+β1xi)}22σ2]\begin{align} f(y_i; \beta_0, \beta_1, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}}\exp{[- \frac{\{y_i - (\beta_0 + \beta_1 x_i)\}^2}{2\sigma^2}]} \end{align}f(yi​;β0​,β1​,σ2)=2πσ2​1​exp[−2σ2{yi​−(β0​+β1​xi​)}2​]​​

になります．以降，面倒なので f(yi;β0,β1,σ2)f(y_i; \beta_0, \beta_1, \sigma^2)f(yi​;β0​,β1​,σ2) は単に f(yi)f(y_i)f(yi​) とします．最尤法とは，与えられた確率密度関数の元で，与えられたデータが得られる確率 (もっともらしさ; 尤度) の対数 (対数尤度関数) が最大になる変数 (この場合はβ0,β1\beta_0, \beta_1β0​,β1​)を求める方法です．

今，yiy_iyi​ は互いに独立だったのでその尤度は
